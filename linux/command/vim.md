1. vim 单文件替换方法   
单文件中使用 :s 命令进行替换操作    
替换操作会涉及几个常用的标记  
% 所有行  g 单行内所有匹配项  i 忽略大小写  c  确认是否替换    
关键词 old 替换为 new  
:s/old/new              将光标所在行第一个 old 替换为 new  
:s/old/new/g            将光标所在行所有 old 替换为 new  
:%s/old/new/gc          全文执行替换,询问是否替换  
:3,10s/old/new/gic      将第3行到第10行内容替换，忽略大小写，且每个询问是否替换
